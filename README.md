# Phishing Awareness Survey — Collected Data

This repository contains the anonymized survey response data collected 
for the research paper:

**"How aware are different groups of people about the term phishing"**  
Purva Ghanate, Technische Hochschule Mittelhessen, 2026

## Contents
- `Ghanate_Phishing_Data_Clean.xlsx` — Anonymized responses from 72 participants

## About the Survey
The survey measured phishing awareness across demographic and occupational 
groups using a structured online questionnaire. It included self-assessment 
questions on familiarity and confidence, as well as an 8-question 
scenario-based phishing detection quiz.

## Column Reference
| Column | Description |
|--------|-------------|
| AN01 | Familiarity with phishing (1–5) |
| AN02 | Confidence in identifying phishing (1–5) |
| AN05 | Frequency of receiving suspicious messages |
| BI01 | Age group |
| BI02 | Gender |
| BI04 | Cybersecurity training received (Yes/No) |
| BI05 | Occupation category |
| ME01–ME08 | Quiz question responses |
| TIME_SUM | Total time spent on survey |
